# 📈 Stock Portfolio Tracker — CodeAlpha Task 2

A professional stock portfolio tracker built in Python with CSV export functionality.

## Features
- View 8 available stocks with live hardcoded prices (AAPL, TSLA, GOOGL, MSFT, AMZN, META, NFLX, NVDA)
- Add stocks with quantity to your portfolio
- Remove stocks from portfolio
- View formatted portfolio table with total investment value
- Save portfolio to `portfolio.csv` with timestamp

## How to Run
```bash
python stock_tracker.py
```
> No external libraries needed — uses Python built-ins only.
